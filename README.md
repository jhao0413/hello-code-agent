## About

Hello Code is a customized version of Neovate Code, an AI-powered coding agent CLI. This project builds upon the excellent foundation of Neovate while adding custom configurations and features.

### Upstream

- Original Project: [neovateai/neovate-code](https://github.com/neovateai/neovate-code)
- Website: [neovateai.dev](https://neovateai.dev)

## Quick Start

```bash
cd hello-code
# Edit .env with your API key
bun index.ts
```

## Syncing with Upstream

```bash
git fetch upstream
git merge upstream/master
```
